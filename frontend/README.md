# SoundLab Frontend

React + Vite app for the SoundLab home studio workflow.

**Full setup, env vars, and deploy steps:** [../docs/frontend.md](../docs/frontend.md)

## Quick start

```bash
npm install
cp .env.example .env.local
npm run dev
```

Set `VITE_API_URL` in `.env.local` to your backend (default `http://127.0.0.1:8000`).
